## 推荐的启动方式

### 首次启动
1. 初始化数据库：`init-db.bat`
2. 初始化钱包系统：`init-wallet.bat`
3. 启动后端：`start-server-now.bat` 或 `cd errand-back && start-server.bat`
4. 在微信开发者工具中打开 errand-front 目录

### 日常开发
- 启动后端：`start-server-now.bat` 或 `cd errand-back && npm start`
- 管理后台：打开 `errand-admin/index.html`
