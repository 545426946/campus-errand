# 快速开始指南

## 🚀 5分钟快速启动

### 第一步：配置数据库

1. 确保MySQL服务已启动
2. 运行配置脚本：
```bash
配置前后端数据交互.bat
```

或手动执行：
```bash
cd errand-back
mysql -u root -p < database/schema.sql
mysql -u root -p < database/seed.sql
mysql -u root -p < database/insert-orders.sql
```

### 第二步：启动后端

```bash
cd errand-back
npm install
npm run dev
```

后端将在 `http://localhost:3000` 启动

### 第三步：启动前端

1. 打开微信开发者工具
2. 导入项目：选择 `errand-front` 目录
3. 在"详情 > 本地设置"中勾选：
   - ✅ 不校验合法域名
   - ✅ 不校验安全域名
4. 编译运行

### 第四步：测试功能

1. 点击"微信快速登录"
2. 查看首页订单列表（数据来自数据库）
3. 点击订单查看详情
4. 尝试接单、发布订单等功能

## 🔍 验证数据交互

### 1. 查看数据库
```sql
USE errand_platform;
SELECT * FROM orders ORDER BY created_at DESC LIMIT 5;
```

### 2. 查看网络请求
在微信开发者工具中：
- 打开"调试器" > "Network"
- 查看API请求和响应

### 3. 测试API
使用Postman或curl测试：
```bash
# 登录
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"code":"test_code"}'

# 获取订单列表（需要token）
curl -X GET http://localhost:3000/api/orders \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## ❓ 常见问题

### 后端启动失败
```bash
# 检查端口占用
netstat -ano | findstr :3000

# 检查Node.js版本
node -v  # 需要14+
```

### 数据库连接失败
1. 检查MySQL服务是否启动
2. 检查 `.env` 文件配置
3. 确认数据库已创建

### 前端请求失败
1. 确认后端服务已启动
2. 检查 `utils/config.js` 中的API地址
3. 查看控制台错误信息
