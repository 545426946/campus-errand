## 样式优化

为所有输入框添加了 `::placeholder` 样式，确保提示文字清晰可见：

```css
.input::placeholder,
.textarea::placeholder,
.form-input::placeholder,
.form-textarea::placeholder {
  color: #999;
  font-size: 26rpx;
}
```

## 优化效果

1. **节省空间**：移除独占一行的标签，输入框更紧凑
2. **清晰提示**：placeholder 包含完整的字段说明和输入提示
3. **必填标识**：使用 `*` 标记必填字段
4. **图标增强**：部分字段使用 emoji 图标（📍📌📱💰）增强视觉识别
5. **用户友好**：用户输入时 placeholder 自动消失，不占用实际内容空间

## 注意事项

- 性别选择和图片上传等特殊组件保留了标签，因为它们不是简单的文本输入
- 所有 placeholder 都使用了清晰的中文说明
- 必填字段在 placeholder 开头使用 `*` 标记
